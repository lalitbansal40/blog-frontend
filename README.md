## Start with AllTypeBlogs

```bash

## Step 1

git clone https://github.com/frontendweb3/AllTypeBlogs.git

## Step 2
cd AllTypeBlogs

## Step 3
pnpm install
# or
yarn install

## Step 4
pnpm dev
# or
yarn dev
```
